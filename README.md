# Dental AI No-Show Prediction

AI-powered SaaS for predicting and reducing missed dental appointments.

## 🚀 Features
- No-show risk prediction using Machine Learning
- Automatic risk score updates in MySQL
- FastAPI prediction endpoint
- Streamlit dashboard for clinic visualization

## 🧠 Tech Stack
- Python
- FastAPI
- MySQL
- Scikit-learn
- Streamlit

## 🎯 Goal
Help dental clinics reduce lost revenue from missed appointments through predictive intelligence.
